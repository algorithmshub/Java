# Reverse An Integer

Write a function to reverse an integer using numeric operators and without using any arrays or other data structures.

The signature of the function is `int f(int n)`

| if the input integer is | return |
|:-------------|:-------------|
| 1234 | 4321 |
| 12005 | 50021 |
| 1 | 1 |
| 1000 | 1 |
| 0 | 0 |
| -12345 | -54321 |
